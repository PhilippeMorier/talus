# Frontend

- `nx serve frontend`
- `nx build frontend --base-href BASE_PATH_PLACEHOLDER --progress=false`
- `nx test frontend`
  - `--watch`
  - `--codeCoverage`
- `nx e2e frontend-e2e` (see
  [builder](https://github.com/nrwl/nx/blob/master/docs/angular/api-cypress/builders/cypress.md))
  - `--watch`
  - `--headless`
- `nx lint frontend`
